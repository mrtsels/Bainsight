# Think-Cell 图表功能完全中文教程

> 本教程基于 think-cell 官方手册编制，涵盖所有图表类型、核心操作和数据管理功能。
> 适用场景：Bain 案例竞争、商业演示、数据报告

---

## 一、插入图表

在 PowerPoint 功能区找到 **Insert（插入）选项卡 → think-cell 组 → Elements（元素）按钮**，选择所需的图表类型。插入时会出现一个矩形框，拖动可调整图表大小和位置，图表会自动吸附对齐现有对象。

**支持的图表类型（Elements 菜单中对应图标）：**

| 类型                   | 用途                                   |
| ---------------------- | -------------------------------------- |
| 柱状图 / 条形图        | 比较类别或展示数据随时间变化           |
| 瀑布图（Bridge Chart） | 展示初始值经过增减后的变化过程         |
| 折线图 / 轮廓图        | 追踪趋势或连续数据（旋转90°为轮廓图） |
| 饼图 / 环形图          | 展示各部分占整体的比例                 |
| 散点图                 | 比较两个维度或分析相关性               |
| 气泡图                 | 展示数据点大小（第三维度）             |
| 面积图                 | 追踪数据系列随时间的变化及累积贡献     |
| 梅尔科图（Mekko）      | 二维堆叠柱图，列宽表示类别相对大小     |
| 组合图                 | 在同一图表中组合柱状和折线             |
| 甘特图（Gantt）        | 项目管理时间线                         |

图表插入后自动处于选中状态（蓝色高亮边框），单击图表即可选中。

## 二、数据输入与数据表

### 2.1 元素数据表（Element Datasheet）

选中图表 → 点击右下角 **Open Datasheet** 按钮，或直接双击图表，打开内置 Excel 数据表。新图表插入后数据表会自动打开。

**核心规则：只输入原始数字，think-cell 自动完成所有计算和格式。**

- 使用 `Tab` 键横向移动，`Enter` 键跳到下一行
- 数据表中的数值变化会**实时反映到图表**
- 支持添加/删除行列、公式输入、TCROUND 四舍五入函数等完整 Excel 功能

#### 绝对值与相对值切换

选中整列或某单元格 → 数据表工具栏 → **Add or Remove % Formatting**，即可在绝对值与百分比之间切换。think-cell 默认将每列之和视为 100%，可在 `100%=` 行手动指定基准值。

#### 行列布局调整

数据表工具栏 → **Edit Data Layout**，可添加或删除以下可选行/列：

- 第一行：类别标签（category labels）
- 第一列：系列标签（series labels）
- `100%=` 行：手动指定百分比基准值

#### 转置数据表

工具栏 → **Transpose Rows and Columns**，行列互换（影响柱状图与条形图布局的切换）。

### 2.2 幻灯片工作簿（Slide Workbook）

每张幻灯片都有自己完整的内置 Excel 工作簿，可同时管理页面上所有图表和数据驱动元素的数据。支持公式、条件格式等复杂 Excel 功能。

### 2.3 从外部 Excel 链接数据

1. 在 Excel 中选中数据区域（含类别标签行和系列标签列）
2. Excel think-cell 工具栏 → **Link to PowerPoint → Chart**
3. PowerPoint 自动启动，点击幻灯片放置图表

- 链接的图表旁会显示来源 Excel 文件名称
- 也可点击 **To Existing Element** 将数据链接到已有图表
- 可在 Excel 中使用单元格引用将数据编译到链接框架
- 隐藏的行列数据不会出现在图表中

### 2.4 从 Tableau 链接

可连接 Tableau 数据源，图表和数据表会自动保持同步更新。

### 2.5 数据管理（Data Links 对话框）

**Tools → Data Links** 可管理大量链接元素：查看所有链接、更新数据、恢复上一次状态、在不同数据源之间切换。

## 三、标签操作

think-cell 提供**自动标签布局**，智能选择最优位置（居中、偏移、引线等）。

### 标签类型一览

| 标签类型         | 支持图表               | 添加/删除方式                     |
| ---------------- | ---------------------- | --------------------------------- |
| Category（类别） | 柱、条、折线、面积     | 右键 → Add/Remove Category Label |
| Series（系列）   | 柱、条、折线、面积     | 右键 → Add/Remove Series Label   |
| Segment（段）    | 柱、条                 | 右键 → Add/Remove Segment Label  |
| Point（点）      | 折线、面积、散点、气泡 | 右键 → Add/Remove Point Label    |
| Total（合计）    | 柱、条、面积           | 右键 → Add Total                 |
| Grand Total      | 柱、条、梅尔科、组合图 | 右键 → Add Grand Total           |
| Tick（刻度值）   | 数值轴                 | 右键数值轴 → Add Tick Labels     |
| Title（标题）    | 轴                     | 右键轴 → Add Title               |

### 3.1 自动与手动标签位置

- 标签自动选择最优位置
- 选中标签后若右下角出现拖拽图标，说明有多个可选位置，拖动即可手动调整
- 手动放置后标签位置固定，不再随图表变化自动更新
- 拖回目标区域或右键 → **Enable Automatic Placement** 可恢复自动模式
- 支持多标签同时拖动（Ctrl+点击多选）

### 3.2 标签文本编辑

选中标签直接输入可覆盖原有文本（包含数据字段）。按 `F2` 进入编辑模式可保留原有数据字段。数据字段以"单个字符"形式存在，随数据源自动更新。覆盖数据字段后标签旁出现红色感叹号，点击感叹号或通过 **Label Content** 菜单可恢复联动。

### 3.3 标签内容格式（Label Content）

在 mini toolbar 的 Label Content 下拉菜单中选择显示内容：绝对值、百分比、两者同时显示，或排序标签（Ordinal，按数值大小显示排名）。

### 3.4 数字格式

在 mini toolbar 数字格式框中输入示例数字即可设定格式（实际数值不影响，只影响格式）：

- 小数位数：`1.2` → 1位小数
- 千分位分隔符：`,` 或 `.` 或空格
- 前后缀：`$1.2M`、`EUR -1.234`、`(1.2M)`（括号表示负值）
- 有符号数字：在正数前加 `+` 如 `+1,234`
- 数量级控制：直接在 mini toolbar 选择 `×10⁶` 等

### 3.5 日期格式代码

| 代码              | 说明                | 示例                                     |
| ----------------- | ------------------- | ---------------------------------------- |
| `d/dd`          | 日（单位/两位）     | `5` / `05`                           |
| `ddd/dddd`      | 星期（缩写/全称）   | `Mon` / `Monday`                     |
| `w/ww`          | 周数（单位/两位）   | `7` / `07`                           |
| `M/MM/MMM/MMMM` | 月份                | `9` / `09` / `Sep` / `September` |
| `Q/QQ`          | 季度（阿拉伯/罗马） | `4` / `iv`                           |
| `yy/yyyy`       | 年份                | `24` / `2024`                        |
| `\`             | 转义自定义字符      | `\QQ` → `Q3`                        |
| `"..."`         | 自定义文本          | `"Quarter" QQ` → `Quarter iv`       |
| `¶`            | Alt+Enter 换行符    |                                          |

## 四、数值轴（Value Axis）

右键图表 → **Chart → Add Value Axis**，或在已有轴右键添加标题、刻度标签、刻度线、网格线、轴线。

### 4.1 调整数值轴

- **手动缩放**：拖动轴两端把手，`Alt+拖动`保持手动模式不恢复自动
- **手动刻度间距**：拖动中间把手，会跳到支持的刻度间距
- **恢复自动**：拖动直到 tooltip 显示 "Automatic"，或右键 → **Fit Scale To Data**
- **对数/线性切换**：右键 → **Set Logarithmic Scale / Set Linear Scale**
- **轴类型切换**：mini toolbar → **Label Content → %**，堆叠柱图自动变为 100% 堆叠柱图
- **倒转轴方向**：散点图、气泡图、折线图支持（右键轴 → **Values in descending order**）

### 4.2 轴断裂（Value Axis Break）

当大 segment 值和小 segment 值差异悬殊时使用。右键 segment 或轴 → **Add Break**：

- 拖动断裂线调整压缩范围
- 支持直线性断裂（straight）和波浪形断裂（wiggle）两种样式
- 断裂位置必须在 segment 边界处
- 断裂对所有共享同一刻度的关联图表同步生效

### 4.3 日期轴（Date Axis）

当类别单元格包含严格递增的年份数字且格式为 Date 时自动识别为日期轴。可混用两位/四位年份格式，选中某标签设置后其他标签自动适配。mini toolbar 提供 **Week Starts On** 下拉菜单选择周起始日。

### 4.4 类别轴（Category Axis）

- **排序方式**：数据表顺序、正序、逆序，或按数值高低升序/降序排列
- **类别间距**：拖动基线把手增加单个类别间距，`Ctrl+拖动`保持总宽度不变
- **类别轴断裂**：在两个柱之间右键 → **Add Category Axis Break**

### 4.5 同步多图表刻度

选中多个图表的 segment（Ctrl+点击）→ 右键 → **Set Same Scale**，将较小图表的刻度扩大到与最大者一致。数据变化时所有关联图表自动同步重缩放。双数值轴图表（折线图/散点图）可分别设置 X 轴和 Y 轴的独立刻度同步。

## 五、图表装饰

### 5.1 差异箭头（Difference Arrows）

选中 segment/point → 右键 → **Add Level Difference Arrow**（两点间相对差异）或 **Add Total Difference Arrow**（柱合计间的差异）：

- 箭头模式可切换：单向箭头+相对值、双向箭头+绝对值、反向箭头+相对值
- 拖动把手设置比较的起点和终点
- 标签可显示绝对差值或相对于 `100%=` 基准的百分比（瀑布图中常用）

### 5.2 CAGR 年复合增长率箭头

选中 segment → 右键 → **Add Compound Growth Arrow**，自动基于数据表日期列计算：

```
CAGR = (结束值/起始值)^(1/年数) - 1
```

- 箭头标签自动显示 CAGR 值，可修改字体和数字格式
- 可选择起点和终点 segment 计算特定区间 CAGR
- 选中多个类别时自动为每对连续类别计算

**系列 CAGR 标签：** 选中系列标签 → mini toolbar → **Label Content → CAGR**，可按系列分别计算 CAGR。

### 5.3 数值线（Value Lines）

右键图表 → **Add Value Line**，显示平行于 X 轴的参考线。自动初始化为算术平均值（堆叠柱图）或加权平均值（梅尔科图），可手动拖动定位，`←→↑↓` 方向键微调。

### 5.4 净线（Net Lines）

选中某 segment/line → mini toolbar 底部 → **Net Line**，显示平行于类别轴的净线，适合比较预期值与实际值或突出趋势偏差。可添加连接线形成阶梯线（step lines），拖动把手使净线形成直角色。

### 5.5 通用连接线（Universal Connectors）

在 think-cell 选项卡中点击通用连接线按钮（双击可连续插入多个，Esc 退出），可连接任意两个可连接点，包括跨图表连接。中部把手可拖出直角形状。

### 5.6 系列连接线（Series Connectors）

在 segment 上右键 → **Add Series Connector**，在该 segment 右上角创建连接线。瀑布图必须使用专用的瀑布连接线（Waterfall Connector）。

### 5.7 100% 指示器

在 100% 堆叠柱图、条形图、面积图和带百分比轴的梅尔科图中可用，自动显示在图表左侧或右侧。

## 六、各图表类型详解

### 6.1 柱状图与条形图

**类型区别：**

| 类型            | 说明                                    |
| --------------- | --------------------------------------- |
| 简单柱状图      | 只输入一行数据即为简单柱状图            |
| 堆叠柱状图      | 多行数据纵向堆叠                        |
| 100% 堆叠柱状图 | 所有柱加总到 100%（可超过或不足）       |
| 簇状柱状图      | 堆叠基础上各 segment 并排排列           |
| 蝴蝶图          | 两个条形图背靠背（旋转180°，相同刻度） |

**组合图**：选中某 segment → mini toolbar → **Line**，将其转换为折线；反之亦然。

**堆积簇状柱图**：插入堆叠柱图 → 拖动柱宽把手到 0% 无间隙 → 在基线上拖动插入类别间隙（每个间隙重复一次）。

**列宽调整**：拖动 segment 中间把手，tooltip 显示间隙宽度百分比（50% 表示间隙是柱宽的一半）。所有柱始终等宽。

**片段时间排序**：在 mini toolbar 的 **Segment Order** 菜单中可按数据表顺序、正序、逆序、升序/降序值排列。

### 6.2 折线图与轮廓图

旋转 90° 即为轮廓图（profile chart）。

- **线条方案**：mini toolbar → **Line Scheme**，控制整体线条外观
- **平滑线**：右键 → **Set to Smooth Line**
- **插值**：缺失值默认线性插值，可在系列右键 → **Do not Interpolate** 禁用
- **次要 Y 轴**：选中某条线 → 右键 → **Set to Right/Left Axis**
- **标记形状**：选中数据点 → mini toolbar 选择形状
- **误差线**：创建足球场图的关键工具

**足球场图制作步骤：**

1. 创建轮廓图（向右旋转的折线图）
2. 在 datasheet 中输入最低值和最高值
3. 选中两条线 → 右键 → **Add Error Bars**
4. 将误差线格式设为粗线（6pt）
5. 右键图表 → **Add Gridlines**，全选后设置为浅色

### 6.3 面积图

本质上是堆叠折线图，面积表示累积值。堆叠面积图每个面积代表一个系列的累积值。100% 堆叠面积图将总值归一化为 100%。支持 Add Total（合计标签）、Add Point Label（点标签）、线性插值（Interpolate）。

**面积排序**：mini toolbar → **Area Order** 菜单中选择堆叠顺序。

### 6.4 瀑布图（Waterfall Charts）

**数据输入规则：**

- 正值：向上 segment
- 负值：向下 segment
- `e`（equals）：自动拉伸至图表总高度的合计 segment
- 支持从右向左反向推算（首列输入 `e`）

**瀑布连接线：**

- 拖动连接线把手改变连接方式
- 右键 → **Remove Waterfall Connector** 删除连接（重新开始累加）
- 右键 → **Add Waterfall Connector** 添加连接
- 当正值和负值在同一列时，数学求和决定连接位置
- 若柱不接地，右键 → **Set to Baseline** 强制接地

**片段时间排序**：mini toolbar → **Segment Order** 菜单。

### 6.5 饼图与环形图

本质上是只有一个类别的 100% 堆叠柱图。

- 拖动四个把手旋转饼图（在 0°/90°/180°/270° 自动吸附）
- 拖出饼图可突出某切片
- 环形图孔径把手拖动可调整（拖到 0% 即变为饼图）

**切片顺序**：mini toolbar → **Slice Order** 菜单：顺时针/逆时针数据顺序、升序/降序数值。

### 6.6 散点图与气泡图

**数据表结构：**

- 每行一个数据点
- `X` / `Y` 列：横纵坐标
- `Group` 列：分组（支持批量选择：点击某点 + Shift+移动）
- 气泡图额外有 `Size` 列（默认与面积成比例，可改为按直径比例）

**趋势线（Trendline）：**

右键数据点 → **Add Trendline/Partition** → 选择类型：线性（默认）、幂函数、指数、对数、二次、三次。使用线性回归自动拟合。

**分区线（Partition）：**

- 可手动拖动定位，`Shift+拖动`保持角度约束
- `Ctrl+拖动`复制平行线（适合标记数值区间）
- 拖动趋势线使其变为分区线（不再基于数据点计算）

**气泡图特殊功能：**

- 气泡大小图例：右键 → **Add Bubble Size Legend**
- 重叠时小气泡在前，`Bring to Front` 调整层序
- 默认超过 300 个数据点时禁用标签，可用右键菜单启用

### 6.7 梅尔科图（Mekko Chart）

二维堆叠柱图，柱宽也表示数据维度，无柱间间隙。

**三种变体：**

| 类型             | 图标        | 说明                                       |
| ---------------- | ----------- | ------------------------------------------ |
| 标准梅尔科图     | Mekko       | X轴为数值轴（基线），柱宽表示类别大小      |
| 百分比轴梅尔科图 | 100% Mekko  | Y轴百分比，柱宽表示绝对总量（面积=绝对值） |
| 单位轴梅尔科图   | Units Mekko | 柱宽和高度完全独立，在 X extent 行输入柱宽 |

**山脊（Ridge）：** 右键图表 → **Add Ridge**，突出显示梅尔科图轮廓线。

**其他系列（Other Series）：** 选中若干小 segment → 右键 → **Move to Other Series**，将多个小段合并为一个"其他"系列。

**片段时间排序：** Segments in Sheet Order / Reverse / Descending / Ascending。

## 七、图表扫描器（Chart Scanner）

从屏幕任意位置（网站、PDF、应用）扫描柱状图或条形图的截图，自动识别数据。

**在 PowerPoint 中使用：**

1. 创建图表（不支持散点图/气泡图/单位轴梅尔科图/Gantt图）
2. 打开图表数据表，双击或点击 Open Datasheet
3. 工具栏 → **Extract and Paste Numerical Data**
4. 移动扫描窗口覆盖图表区域（只含图表、轴和标签，排除标题和说明）
5. 扫描完成后点击 **Fill Datasheet**，数据自动填入

**在 Excel 中使用：**

`think-cell → Data → Chart Scanner`，选中目标单元格区域后启动扫描，完成后点击 **Paste Into Excel**。数据可进一步链接到 think-cell 元素。

## 八、图表+数据表组合

在图表旁自动生成关联的数据表格，数据可在图表和数据表之间自由移动。

**启用：** `Insert → think-cell → Tools → Enable Data Tables for New Charts`

**支持类型：** 柱状图、条形图、折线图、轮廓图、组合图、面积图、瀑布图、梅尔科图

**数据移动方式：**

- 图表上右键 segment/category/series → **Move to Table Row / Column**
- 数据表行/列上右键 → **Move Table Row / Column to Chart**
- 数据表工具栏中选中 → **Move Selected Rows / Columns to Table / Chart**

think-cell 根据图表大小和位置自动调整表格尺寸，可独立移动和调整。

## 九、样式与格式化

### 9.1 填充颜色

选中 segment → mini toolbar → **Fill Color** 选择颜色。选中多个 segment 可批量设置。

### 9.2 匹配数据源填充（Use Datasheet Fill on Top）

1. 在 Excel 中设置单元格背景色
2. 选中图表 → mini toolbar → **Color Scheme → Use Datasheet Fill on Top**
3. Excel 单元格填充色自动应用到对应 segment
4. 若单元格无填充色，该 segment 使用图表颜色方案

### 9.3 颜色方案与填充顺序

- 选中图表 → mini toolbar → **Color Scheme**
- 添加/移除系列时，填充方案自动循环应用
- **Stacked Fill Order: Bottom to Top** 可反转填充顺序

### 9.4 匹配 Excel 条件格式

1. 在 Excel 中选中链接单元格（含链接到元素的单元格）
2. `Home → Styles → Conditional Formatting` 设置条件格式
3. 在 PowerPoint 中选中链接元素 → mini toolbar → 相应格式选项

> 限制：不支持 Data Bars 和 Icon Sets 条件格式；渐变色不支持；Excel 中修改条件格式后需改动任意数据值或重启 Excel 才能更新。

## 十、数据四舍五入（TCROUND）

think-cell 内置 Excel 四舍五入工具，解决手工四舍五入导致合计不一致的问题。

### 10.1 问题与原理

标准 Excel `ROUND()` 单独对每个单元格四舍五入，累积误差大。例如：3.7+4.5+3.7=12，四舍五入后 4+5+4=13（多出1）。

TCROUND 从全局优化出发，选择最少量的值向反方向舍入，使合计精确等于舍入后各项之和，同时偏差最小化。

### 10.2 函数与用法

在 Excel 中选中单元格范围 → **Formulas → think-cell round**：

| 函数                  | 说明                             |
| --------------------- | -------------------------------- |
| `TCROUND(x, n)`     | 自动选择最小误差方向（优先使用） |
| `TCROUNDUP(x, n)`   | 强制向上舍入（远离零）           |
| `TCROUNDDOWN(x, n)` | 强制向下舍入（趋向零）           |
| `TCROUNDNEAR(x, n)` | 强制四舍五入到最近倍数           |

**精度参数 `n` 示例：**

| n      | 效果     | 示例输入 → 输出 |
| ------ | -------- | ---------------- |
| `0`  | 整数     | 17.4 → 17       |
| `1`  | 0.1      | 1.018 → 1.02    |
| `2`  | 0.01     | 54.6 → 54.60    |
| `-2` | 100      | 1234 → 1200     |
| `50` | 50的倍数 | 1234 → 1250     |

### 10.3 重要限制

- **TCROUND 必须是最外层函数**：
  - ✓ `=TCROUND(A1+SUM(B1:E1), 1)`
  - ✗ `=TCROUND(A1,1)+TCROUND(SUM(B1:E1),1)`
- **避免在 TCROUND 内使用随机函数**（`RAND()` 等），否则计算会出错
- 涉及乘法（非常量乘以 TCROUND 结果）和复杂函数时，**不保证存在一致解**，会返回 `#NUM!`
- 当问题无解时，尝试：使用更精细的精度参数、仅用 `+`、`-`、`SUM` 连接公式、所有 TCROUND 使用相同精度参数
