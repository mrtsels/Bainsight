## 设计规范总结

### 页面结构原则
- **每页只传达一个核心观点**
- **模块 = 论点 + 论据展示**，紧密耦合
- **论据形式**：图、表、流程图、数据卡片、对比图等
- **文字精简**：能用图表绝不用文字

### 模块化标准结构
```
┌─────────────────────────────────┐
│  模块N: [模块主题]              │
├─────────────────────────────────┤
│  论点: [核心观点]                │
│  ────────────────────────────   │
│  论据: [图表/表格/流程/数据]     │
└─────────────────────────────────┘
```

### 视觉规范
| 元素 | 规范 |
|------|------|
| 主色 | 贝恩红 #E60000 + 黑色系 |
| 字体 | 无衬线，层级分明 |
| 一页一核心 | 每页一个Action Title |
| 图表优先 | 表格、图示、数据卡片 |

### 一致性校验
- 每页附数据来源
- 关键数字加粗
- 颜色语义统一（红=警示/止血，绿=增长/最优）
