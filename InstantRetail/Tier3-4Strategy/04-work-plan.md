## WS-A: 城市组合决策

### AX-A1: 10城市比较财务分析
- **问题**: 哪些城市单位经济最好/最差？为什么？
- **方法**: 10城横截面对比——AOV、履约成本、补贴、成本/AOV、高价值用户占比、品类结构
- **数据**: 01-problem-definition.md 中的10城数据集
- **数据状态**: ✅ 完备；⚠️ **Case PDF 可能有额外数据，待确认**
- **输出**: 城市排名表 + 2×2矩阵（AOV × 成本/AOV）
- **假设**: C3/C7/C8为最佳；C9/C10/C4为结构性劣势
- **优先级**: P1
- **依赖**: 无

### AX-A2: 三类城市盈利路径 + 威海模式洞察
- **问题**: (1) 泉州/惠州/威海各自靠什么实现EBIT+？路径是否相同？(2) 威海模式能否复制到其他城市？
- **方法**: 各城市单位经济分解；识别主要EBIT驱动因子；对比威海与其他城市的驱动因素特征；按"威海适配度"对城市排序
- **数据**: 01-problem-definition.md；⚠️ Case PDF 可能有额外数据
- **数据状态**: ✅ 完备（**合并AX-A5核心洞察，不再独立分析**）
- **输出**: 3城对比表+EBIT驱动归因；各城市"威海适配度"排名；AOV提升路径
- **假设**: 泉州靠大超深度；惠州靠规模；威海靠高AOV——三条不同路径。威海模式在C7（惠州）和C1（徐州）部分可复制；C9/C10短期不可复制
- **优先级**: P1
- **依赖**: AX-A1（可并行）

### AX-A3: 选择性收缩成本收益分析
- **问题**: 退出或收缩C9/C10/C4/C5能节省多少固定成本和补贴？
- **方法**: 估算各城市年亏损；建模退出节省（固定成本分摊减免 + 补贴节省）；区分部分退出/完全退出的敏感性
- **数据**: 01-problem-definition.md（年亏损估算）；⚠️ Case PDF 可能有更精确的固定成本数据
- **数据状态**: ✅ 完备（但亏损数为估算，非实际）
- **输出**: 节省区间表（部分/完全退出）；建议退出时序
- **假设**: 完全退出2个最差城市节省~3,000万/年；部分退出4个评估城市节省~2,000万
- **优先级**: P1
- **依赖**: AX-A1（可并行）

### AX-A4: 3+3+4城市分组EBIT情景模型
- **问题**: 3+3+4分组在保守/基准/乐观假设下，3年EBIT路径如何？
- **方法**: 三情景财务模型；第1年：3城EBIT+；第2年：+3城；第3年：整体EBIT+。关键驱动：AOV改善、补贴压降、部分收缩节省
- **数据**: AX-A1/02/03输出；01-problem-definition.md基础数据
- **数据状态**: ✅ 完备（亏损数为估算）
- **输出**: 各情景3年EBIT桥；盈亏平衡时点
- **假设**: 即使保守情景，3城聚焦策略也能在第2年实现EBIT+
- **优先级**: P1
- **依赖**: AX-A1、AX-A2、AX-A3 — **门控分析（Day 1产出）**

## WS-B: 竞争约束边界

> **执行方**: Partner 直接执行（不通过团队）。执行方：美团/京东年报、分析师报告（高盛/德银）、行业新闻检索。
>
> **目标**: 即使只找到一个真实数据点（如"美团2024年披露全国单均补贴X元"），也比纯假设更有价值。

### AX-B1: 美团公开财务及三四线战略研究
- **问题**: 美团在三四线城市的单均补贴和研发支出是多少？其三四线战略是什么？
- **方法**: 检索美团2024年报、投资者PPT、分析师电话纪要；搜索"美团四五线城市即时零售补贴"
- **数据来源**: 美团投资者关系（hkdata.com）、彭博、高盛/德银分析师报告
- **具体目标数据点**（按优先级）:
  1. 美团披露的到家业务营销+补贴支出/单（年报或投资者日）
  2. 美团研发/营销在三四线城市的分配（年报分部披露）
  3. 财报电话会中对四五线城市即时零售战略的明确表述
- **数据状态**: 🔍 **Day 1 启动（Partner 直接执行）**
- **输出**: 关键发现备忘录（补贴下限估算、三四线战略描述）
- **置信度**: 中 — 仅有全国数据，无城市级披露；若找到任何真实数据点则升级
- **优先级**: P1
- **依赖**: 无

### AX-B2: 京东公开财务及京东到家战略研究
- **问题**: 京东在低线城市的即时零售战略是什么？京东到家如何竞争？
- **方法**: 检索京东2024年报、京东到家投资者材料、艾瑞/易观即时零售市场份额报告
- **数据来源**: 京东投资者关系、艾瑞、易观、iiMedia
- **具体目标数据点**（按优先级）:
  1. 京东到家来自二三线城市的收入贡献（若有披露）
  2. 京东在低线城市即时零售扩张的披露战略
  3. 艾瑞/易观/iiMedia 关于三四线城市即时零售市场份额数据
- **数据状态**: 🔍 **Day 1 启动（与AX-B1并行）**
- **输出**: 关键发现备忘录（低线城市竞争烈度、京东到家差异化定位）
- **置信度**: 中
- **优先级**: P1
- **依赖**: 无（与AX-B1并行）

### AX-B3: 竞争烈度情景分析
- **问题**: 不同竞争情景如何约束即时零售的补贴压降空间？
- **方法**: 综合AX-B1+B2发现，构建3种竞争情景（高/中/低）。高情景以真实美团数据为锚（若找到）；中/低情景以估算为基准。每种情景下：估算最大可行补贴压降幅度、识别应对选项、标注触发点
- **数据**: AX-B1、AX-B2研究输出
- **数据状态**: 待AX-B1/B2（**Day 2产出**）
- **输出**: 3×3情景矩阵（竞争烈度 × 内部改善）；各情景补贴下限；关键触发点
- **假设**: 即使高竞争情景，若仅针对低价值用户，仍有一定补贴压降空间
- **优先级**: P1
- **依赖**: AX-B1、AX-B2 — **门控分析（Day 2产出）**

## WS-C: 补贴精准化空间

### AX-C1: 补贴效率诊断
- **问题**: 当前4.2元/单的补贴在各城市和用户质量层级如何分布？
- **方法**: 各城市补贴/单 vs. 高价值用户占比的横向对比；识别补贴效率最差的城市
- **数据**: 01-problem-definition.md（各城市补贴和用户占比）
- **数据状态**: ✅ 完备
- **输出**: 补贴效率排名表；各城市"浪费率"代理指标
- **待验证发现**: C10补贴最高（4.9元）+高价值用户最低（15%）=效率最差
- **优先级**: P1
- **依赖**: 无

### AX-C2: 高价值用户改善潜力分析
- **问题**: 威海（26%）vs. 赣州（15%）的差距告诉我们什么改善空间？
- **方法**: 对比威海与赣州的城市和用户特征；识别11pp差距的结构性驱动因子（品类结构？大超深度？城市人口特征？补贴精准度？）
- **数据**: 01-problem-definition.md；⚠️ Case PDF 可能有用户行为队列数据
- **数据状态**: ⚠️ 部分完备 — 有断面数据，无纵向数据；Case PDF 可能有补充
- **输出**: 差距分解；定性改善路径；改善上限估算
- **假设**: 11pp差距部分由品类结构和大超深度驱动——两者均可在12个月内改善
- **优先级**: P1
- **依赖**: AX-A1（可并行）

### AX-C3: 补贴重分配情景分析
- **问题**: 将补贴从低价值用户向高价值用户重分配，在各竞争情景下的财务影响是什么？
- **方法**: 建模12个月补贴重分配的三种情景（保守/基准/乐观）。Day 2 以初步竞争假设启动：保守=低价值用户补贴减少10%；基准=减少20%+留存激励；乐观=完全重分配。Day 3 待 AX-B3 竞争天花板锁定后最终输出
- **数据**: AX-C1、AX-C2输出；AX-B3竞争天花板输入
- **数据状态**: ⚠️ Day 2 可提前建模（初步假设）；Day 3 最终锁定
- **输出**: 各情景补贴节省区间；高价值用户占比轨迹
- **注**: 将以"方向性估算+明确假设"呈现，而非精确预测
- **优先级**: P1
- **依赖**: AX-C1、AX-C2（Day 2 可提前）；**AX-B3**（Day 3 最终门控）
